# SKILLS DE CALISTENIA - MAPA POR NIVELES
## De Principiante a Experto (12 Niveles)

---

## NIVEL 1: Novato Absoluto (0-3 meses)
**Objetivo:** Aprender movimientos básicos, crear hábito
**Frecuencia:** 3x semana

### Habilidades disponibles:
- **Push-up en pared** → Meta: 3x15
- **Push-up inclinado** → Meta: 3x12  
- **Sentadilla con apoyo** → Meta: 3x20
- **Plancha (rodillas)** → Meta: 30s
- **Australian Row** → Meta: 3x10
- **Puentes de glúteo** → Meta: 3x15

### Evaluación para subir a Nivel 2:
- [ ] 10 flexiones inclinadas seguidas
- [ ] 20 sentadillas seguidas
- [ ] 30 segundos plank
- [ ] 5 Australian rows

---

## NIVEL 2: Principiante I (3-6 meses)
**Objetivo:** Dominar peso corporal básico
**Frecuencia:** 3x semana

### Habilidades disponibles:
- **Flexión estándar** → Meta: 3x15
- **Flexión rodillas** → Meta: 3x12
- **Sentadilla profunda** → Meta: 3x25
- **Zancada estática** → Meta: 3x10/pierna
- **Plancha completa** → Meta: 45s
- **Puente una pierna** → Meta: 3x10

### Skill técnico desbloqueado:
- **🎯 Handstand preparación:** Pike push-ups → Meta: 3x8

### Evaluación para subir a Nivel 3:
- [ ] 10 flexiones completas perfectas
- [ ] 30s plank completo
- [ ] 5 pike push-ups
- [ ] 15 sentadillas profundas

---

## NIVEL 3: Principiante II (6-9 meses)
**Objetivo:** Introducir dominadas y progresiones
**Frecuencia:** 3-4x semana

### Habilidades disponibles:
- **Dominada negativa** → Meta: 3x5 (bajada 5s)
- **Flexión diamante** → Meta: 3x10
- **Flexión ancha** → Meta: 3x12
- **Sentadilla búlgara** → Meta: 3x8/pierna
- **Plancha lateral** → Meta: 20s/por lado

### Skills técnicos desbloqueados:
- **🎯 Handstand Fase 1:** Wall walks → Meta: 3 subidas
- **🎯 L-sit preparación:** L-sit tuck → Meta: 10s

### Evaluación para subir a Nivel 4:
- [ ] 3 dominadas negativas controladas
- [ ] 15 flexiones estándar
- [ ] 10s L-sit tuck
- [ ] 3 wall walks completos

---

## NIVEL 4: Principiante III (9-12 meses)
**Objetivo:** Primera dominada completa, handstand en pared
**Frecuencia:** 3-4x semana

### Habilidades disponibles:
- **Dominada estándar** → Meta: 3x5
- **Dominada asistida con banda** → Meta: 3x8
- **Fondos en banco** → Meta: 3x12
- **Sentadilla pistol asistida** → Meta: 3x5/pierna
- **Dragon flag negativo** → Meta: 3x3

### Skills técnicos desbloqueados:
- **🎯 Handstand Fase 2:** Chest-to-wall hold → Meta: 30s
- **🎯 Muscle-up preparación:** High pull-ups → Meta: 3x5 (pecho a barra)
- **🎯 L-sit:** L-sit completo en suelo → Meta: 5s

### Evaluación para subir a Nivel 5:
- [ ] 1 dominada completa (chin-up o pull-up)
- [ ] 20 flexiones seguidas
- [ ] 20s chest-to-wall handstand
- [ ] 3s L-sit completo

---

## NIVEL 5: Intermedio I (1-1.5 años)
**Objetivo:** Construir volumen, primer muscle-up
**Frecuencia:** 4x semana (Upper/Lower)

### Habilidades disponibles:
- **Dominada supina** → Meta: 3x8
- **Dominada prona** → Meta: 3x5
- **Flexión declinada** → Meta: 3x10
- **Pistol squat** → Meta: 3x3/pierna
- **Fondos paralelas** → Meta: 3x8

### Skills técnicos desbloqueados:
- **🎯 Handstand Fase 3:** Back-to-wall hold → Meta: 20s
- **🎯 Muscle-up Fase 1:** False grip hang → Meta: 20s
- **🎯 Front Lever prep:** Tuck front lever → Meta: 10s
- **🎯 L-sit avanzado:** L-sit 15s + L-sit raises

### Evaluación para subir a Nivel 6:
- [ ] 5 dominadas seguidas
- [ ] 25 flexiones seguidas
- [ ] 1 pistol squat completo
- [ ] 10s tuck front lever

---

## NIVEL 6: Intermedio II (1.5-2 años)
**Objetivo:** Muscle-up, fuerza unilateral
**Frecuencia:** 4x semana

### Habilidades disponibles:
- **Dominada con peso** → Meta: +5kg x5
- **Flexión arquero** → Meta: 3x5/por lado
- **Pistol squat completo** → Meta: 3x5/pierna
- **Fondos anillas** → Meta: 3x6
- **Dominada L-sit** → Meta: 3x3

### Skills técnicos desbloqueados:
- **🎯 Handstand Fase 4:** Freestanding kick-ups → Meta: 5 intents
- **🎯 Muscle-up FASE COMPLETA:** Strict muscle-up → Meta: 1 rep
- **🎯 Front Lever Fase 1:** Advanced tuck → Meta: 15s
- **🎯 Back Lever:** Tuck back lever → Meta: 10s

### Evaluación para subir a Nivel 7:
- [ ] 1 muscle-up completo (barra)
- [ ] 8 dominadas seguidas
- [ ] 3 pistol squats seguidos
- [ ] 5s freestanding handstand

---

## NIVEL 7: Intermedio III (2-2.5 años)
**Objetivo:** Skills intermedios, planche inicio
**Frecuencia:** 4-5x semana

### Habilidades disponibles:
- **Dominada explosiva** → Meta: 3x5 (cadera a barra)
- **Flexión pseudo-planche** → Meta: 3x8
- **Pistol squat con peso** → Meta: +10kg x3
- **Fondos anillas RTO** → Meta: 3x5 (RTO = rings turned out)
- **Muscle-up L-sit** → Meta: 2 reps

### Skills técnicos desbloqueados:
- **🎯 Handstand Fase 5:** Freestanding hold → Meta: 10s
- **🎯 Planche Fase 0:** Planche lean → Meta: 30s
- **🎯 Front Lever Fase 2:** One-leg front lever → Meta: 10s/pierna
- **🎯 Human Flag prep:** Side body holds → Meta: 20s

### Evaluación para subir a Nivel 8:
- [ ] 3 muscle-ups seguidos
- [ ] 10s freestanding handstand
- [ ] 15s planche lean
- [ ] 5s one-leg front lever

---

### Skills técnicos desbloqueados:

- **🎯 Planche Fase 1:** Tuck planche → Meta: 10s
- **🎯 Front Lever Fase 3:** Straddle front lever → Meta: 5s
- **🎯 Back Lever completo:** Full back lever → Meta: 10s
- **🎯 Human Flag Fase 1:** Tuck flag → Meta: 5s

### Evaluación para subir a Nivel 9:
- [ ] 5 muscle-ups anillas
- [ ] 3s tuck planche
- [ ] 3s straddle front lever
- [ ] 10m handstand walk

- **🎯 Planche Fase 2:** Straddle planche → Meta: 5s
- **🎯 Front Lever Fase 4:** Full front lever → Meta: 3s
- **🎯 Human Flag Fase 2:** One-leg flag → Meta: 5s

**Objetivo:** Planche completa, one-arm pull-up
**Frecuencia:** 5-6x semana

### Habilidades disponibles:
- **Dominada one-arm** → Meta: 1 rep

- **Nordic curl completo** → Meta: 3x3

### Skills técnicos desbloqueados:

- **🎯 Front Lever Fase 5:** Front lever pulls → Meta: 3 reps
- **🎯 Human Flag completa:** Full flag → Meta: 5s
- **🎯 Victorian completo:** Victorian cross → Meta: 3s

### Evaluación para subir a Nivel 11:
- [ ] 1 one-arm pull-up (cualquier agarre)

---

## NIVEL 11: Experto (4-5 años)
**Objetivo:** Combinaciones, peso adicional máximo
**Frecuencia:** 5-6x semana

### Habilidades disponibles:
- **Dominada one-arm con peso** → Meta: +5kg
- **Muscle-up one-arm negativo** → Meta: 1 rep
- **Planche push-ups** → Meta: 3 reps
- **Nordic curl con peso** → Meta: +10kg
- **Dominadas 20+ seguidas**

### Skills técnicos desbloqueados:
- **🎯 Handstand Fase 9:** Press to handstand → Meta: 1 rep
- **🎯 Planche Fase 4:** Planche press → Meta: 1 rep
- **🎯 Front Lever Fase 6:** Front lever row → Meta: 5 reps
- **🎯 Maltese:** Maltese cross → Meta: 3s
- **🎯 Human Flag avanzada:** Flag raises → Meta: 3 reps

### Evaluación para subir a Nivel 12:
- [ ] 3 one-arm pull-ups seguidos
- [ ] 5s full planche
- [ ] 3 freestanding HSPU
- [ ] 5s maltese

---

## NIVEL 12: Master (5+ años)
**Objetivo:** Dominio absoluto, innovación
**Frecuencia:** Personalizado

### Habilidades disponibles:
- **Cualquier variación de one-arm** (supina, prona, neutra)
- **Planche 360°**
- **Muscle-up one-arm**
- **Combinaciones:** Planche → Handstand → Maltese
- **Peso máximo en skills básicos**

### Skills técnicos MASTER:
- **🎯 Handstand Fase 10:** One-arm handstand → Meta: 5s
- **🎯 Planche Fase 5:** Planche 360° / Planche on rings
- **🎯 Victorian máximo:** Weighted victorian
- **🎯 Combinaciones libres:** Crear secuencias

### Este nivel no tiene "final":
- Competencias
- Crear nuevas progresiones
- Entrenar a otros
- Documentar progreso
