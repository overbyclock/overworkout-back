## 1. TIPOS DE SKILLS (Categorías)

Los skills técnicos se dividen en **5 familias** que se desbloquean en diferentes momentos:

### 🎯 FAMILIA A: "El Pino" (Handstand)
**Desbloqueo:** Nivel 3-4 | **Dominio:** Nivel 8-10

Progresión técnica:
```
FASE 1 - CONSTRUCCIÓN BASE (Nivel 3-4)
├── Pike Push-ups (fortaleza)
├── Wall Walks (propiocepción invertida)
├── Chest-to-wall Hold (alineación)
└── Back-to-wall Hold (equilibrio)

FASE 2 - EQUILIBRIO (Nivel 5-6)  
├── Freestanding Holds (suelo/rampa)
├── Weight Shifts (cambios de peso)
├── Tucked hops (saltos controlados)
└── Spotting (con ayuda/spotter)

FASE 3 - MOVIMIENTO (Nivel 7-8)
├── Handstand Walks (caminata)
├── Shoulder Taps (estabilidad)
├── Pirouette Bail (caída controlada)
└── Press negatives (subida controlada)

FASE 4 - ADVANCED (Nivel 9-10)
├── Straddle Press (entrada straddle)
├── Stalder Press (entrada compresión)
├── One-arm progressions
└── HSPU Freestanding

FASE 5 - MASTER (Nivel 11-12)
├── One-arm Handstand
└── Planche Press Handstand (transición)
```

**Integración en entrenamiento:**
- Nivel 3-4: 10min al inicio (skill work)
- Nivel 7-8: 20min dedicados (skill day)
- Nivel 10+: Puede ser el main movement del día

---

### 🎯 FAMILIA B: "El Muscle-Up" 
**Desbloqueo:** Nivel 5 | **Dominio:** Nivel 8-9

No es solo fuerza, es **TIMING + TRANSICIÓN**:

```
FASE 1 - FUERZA BASE (Nivel 4-5)
├── High Pull-ups (pecho a barra)
├── Explosive Pull-ups (cadera a barra)  
├── False Grip hangs (30s)
└── Chest-to-bar holds (isométrico arriba)

FASE 2 - TRANSICIÓN (Nivel 5-6)
├── Muscle-up Negatives (bajada controlada)
├── Band-assisted MU (asistencia progresiva)
├── Transition holds (falso agarre arriba)
└── Dip portion isolation (fondos altos)

FASE 3 - COMPLETO (Nivel 6-7)
├── Strict Muscle-up (sin kip)
├── False Grip MU (agarre falso)
├── Kipping MU (CrossFit style)
└── MU on Rings (estable)

FASE 4 - VARIACIONES (Nivel 8-9)
├── Weighted MU (+10kg)
├── L-sit MU
├── Slow MU (3s transición)
└── MU to Lever (combinación)

FASE 5 - MASTER (Nivel 10+)
├── One-arm MU progressions
└── MU 360° (giro completo)
```

**Detección automática de progreso:**
La app detecta si el usuario puede hacer:
- 10 pull-ups seguidos → Desbloquea "Intentar Muscle-up"
- False grip 30s → Añade ejercicios de transición
- MU negativo 5s → Rutina de muscle-up completo

---

### 🎯 FAMILIA C: "Front Lever" (Palanca Frontal)
**Desbloqueo:** Nivel 6 | **Dominio:** Nivel 9-11

```
FASE 1 - COMPRESIÓN (Nivel 5-6)
├── Tuck Holds (10s, 20s, 30s)
├── Tuck-to-Inverted (dinámico)
├── Advanced Tuck (rodillas alejadas)
└── Ice Cream Makers (tuck, swing)

FASE 2 - EXTENSIÓN PARCIAL (Nivel 7-8)
├── One-leg Extended (10s cada pierna)
├── Straddle FL (10s, 20s, 30s)
├── Straddle FL Rows (fuerza)
└── Full FL negatives (bajada)

FASE 3 - FULL (Nivel 8-9)
├── Full FL Hold (3s, 5s, 10s, 15s)
├── FL Pulls (subir desde dead hang)
├── FL Rows (remo en posición)
└── FL to Inverted (dinámico)

FASE 4 - ADVANCED (Nivel 10-11)
├── Weighted FL (+5kg, +10kg)
├── One-arm FL progressions
├── Victorian Cross (más bajo)
└── FL Muscle-up
```

---

### 🎯 FAMILIA D: "Planche" (Plancha)
**Desbloqueo:** Nivel 7 | **Dominio:** Nivel 10-12

El skill más difícil, requiere **muñecas de acero**:

```
FAse 0 - PREPARACIÓN (Nivel 5-6)
├── Planche Lean (proyección máxima)
├── Pseudo Planche Push-ups
├── Frog Stand (30s)
└── Wrist conditioning (crucial!)

FASE 1 - TUCK (Nivel 7)
├── Tuck Planche (10s, 20s, 30s, 45s)
├── Tuck PL Press (subir desde suelo)
├── Tuck PL Swings (balance)
└── Advanced Tuck (rodillas 90°)

FASE 2 - STRADDLE (Nivel 8-9)
├── Straddle PL (5s, 10s, 15s)
├── Straddle PL Push-ups
├── Straddle PL Press
└── Half-lay (transición)

FASE 3 - FULL (Nivel 10+)
├── Full PL (3s, 5s, 8s)
├── PL Push-ups
├── PL Press
└── Maltese (PL + brazos abiertos)
```

---

### 🎯 FAMILIA E: "Human Flag" (Bandera Humana)
**Desbloqueo:** Nivel 8 | **Dominio:** Nivel 10-11

Skill lateral, único que requiere **agarre vertical**:

```
FASE 1 - FUERZA LATERAL (Nivel 7-8)
├── Side Plank (60s, elevado)
├── Copenhagen Plank
├── Side Body Holds (en barra vertical)
└── Dragon Flags (core relevante)

FASE 2 - POSICIONES PARCIALES (Nivel 8)
├── Tuck Flag (rodillas al pecho)
├── One-leg Flag (una pierna extendida)
├── Straddle Flag (piernas abiertas)
└── Negative Flags (bajada controlada)

FASE 3 - FULL FLAG (Nivel 9-10)
├── Full Flag Hold (3s, 5s, 10s)
├── Flag Raises (subir desde vertical)
├── Flag Press (bajar a vertical)
└── Switch Flags (cambiar lado sin bajar)
```

## 2. SISTEMA DE EVALUACIONES (EXÁMENES)

### Tipos de Evaluaciones:

#### A) EVALUACIÓN DE SUBIDA DE NIVEL (Cada 12 semanas)
**Formato:** Test físico estandarizado de 45-60 minutos

**Estructura:**
```
SECCIÓN 1: FUERZA MÁXIMA (20 min)
├── Máximo pull-ups en 1 set
├── Máximo push-ups en 2 min
├── Máximo dips en 1 set
├── Máximo pistol squats (cada pierna)
└── Máximo L-sit hold

SECCIÓN 2: SKILLS (20 min)
├── Intentar skill objetivo del nivel
├── Tiempo en posición (holds)
└── Video técnico (opcional, para review)

SECCIÓN 3: CONDICIÓN (15 min)
├── Circuito metabolico (ej: 5 rounds de burpees, pull-ups, squats)
└── Tiempo total o reps totales

SECCIÓN 4: RECUPERACIÓN (5 min)
├── Ritmo cardíaco 2 min post-ejercicio
└── RPE (Rate of Perceived Exertion)
```

**Baremos para aprobar:**
- **APROBADO (Sube nivel):** 80% de benchmarks cumplidos
- **CONDICIONAL (Mismo nivel, ajuste):** 60-79% cumplidos
- **NO APROBADO (Repite fase):** <60% cumplidos

---

#### B) SKILL TEST (Evaluaciones específicas de skills)
**Formato:** Micro-test de 10-15 min, se hace cuando el usuario lo solicita o la app detecta progreso

**Ejemplo - Test de Muscle-up:**
```
1. ¿Puedes hacer 10 pull-ups seguidos? [Sí/No]
2. False grip hang máximo tiempo: ___ segundos
3. Intenta muscle-up negativo: [Lo logré / No]
4. ¿Puedes hacer muscle-up asistido? [Sí/No]
5. Intenta muscle-up completo: ___ reps

RESULTADO:
- Si pasas #1 y #2 → Desbloquea "Fase Transición"
- Si pasas #3 → Acceso a "Strict Muscle-up training"
- Si pasas #5 → Skill Muscle-up DESBLOQUEADO + Insignia
```

---

#### C) EVALUACIÓN SEMANAL (Auto-evaluación)
Usuario reporta cada domingo:
- Sensación de progreso (1-10)
- Dolores/molestias (checklist)
- PRs de la semana (nuevos máximos)
- Dificultad de entrenamientos (RPE promedio)

La app ajusta automáticamente:
- Si RPE >8 por 2 semanas → Bajar volumen 10%
- Si no hay PRs en 4 semanas → Cambiar progresión
- Si dolor >5 → Recomendar descanso/doctor

## 4. ESTRUCTURA DE SESIÓN CON SKILL WORK

### TIPO A: Día de Skill Dedicado (Nivel 7+)
```
ENTRENAMIENTO - SKILL DAY "HANDSTAND"

0:00 - 0:10 | CALENTAMIENTO ESPECÍFICO
• Muñecas: 5 min movilidad
• Hombros: dislocaciones, band work
• Core: hollow body activation

0:10 - 0:30 | SKILL PRACTICE (20min block)
• Freestanding holds: 5 x max hold
• Wall work: 3 x HSPU negatives
• Balance drills: shoulder taps, weight shifts
• Nota: Descanso 2-3 min entre intentos

0:30 - 0:35 | TRANSICIÓN / RECUPERACIÓN

0:35 - 1:05 | FUERZA (30min)
• Push: Pseudo planche push-ups 4x8
• Pull: Weighted pull-ups 4x6
• Legs: Pistol squats 4x5

1:05 - 1:15 | CORE (10min)
• L-sit practice
• Dragon flag negatives

1:15 - 1:25 | ENFRIAMIENTO
```

### TIPO B: Skill Integrado (Nivel 4-6)
```
ENTRENAMIENTO - UPPER BODY + SKILL

CALENTAMIENTO (10min)
• General cardio
• Skill prep: 3 x wall walk (subida y bajada)

SKILL BLOCK A (10min) - Técnica
• Muscle-up transition practice
  - 3 sets de: False grip hang 10s + Transition negative

FUERZA (40min)
• Pull-ups: 4x6-8
• Rows: 3x10

SKILL BLOCK B (10min) - Aplicación
• 3 intentos de muscle-up asistido
• Registrar mejor intento

CORE (10min)
• Hollow body holds
• Leg raises
```

## 6. BASE DE DATOS - ESTRUCTURA DETALLADA

```sql
-- Tabla principal de skills
calisthenia_skills (
  id INT PRIMARY KEY,
  name VARCHAR(100),              -- "Muscle-up", "Handstand", "Front Lever"
  family ENUM('handstand', 'muscle_up', 'front_lever', 'planche', 'human_flag', 'other'),
  
  -- Jerarquía
  parent_skill_id INT NULL,       -- Skill previo requerido (NULL si es base)
  unlocks_skill_id INT NULL,      -- Skill que desbloquea este
  
  -- Niveles
  unlock_level INT,               -- Nivel de usuario para desbloquear
  mastery_level INT,              -- Nivel donde se considera "dominado"
  
  -- Progresiones (JSON)
  progressions JSON,              -- [
                                  --   {"step": 1, "name": "High Pull-ups", "req": "3x8"},
                                  --   {"step": 2, "name": "False Grip", "req": "30s"},
                                  --   {"step": 3, "name": "MU Negative", "req": "5s"},
                                  --   {"step": 4, "name": "Strict MU", "req": "1 rep"}
                                  -- ]
  
  -- Benchmarks para desbloqueo
  prerequisite_exercises JSON,    -- {"exercise_id": 45, "min_reps": 10}
  
  -- Recursos
  tutorial_video_url VARCHAR(255),
  common_mistakes TEXT,
  safety_notes TEXT,
  
  -- Metadatos
  difficulty_score INT,           -- 1-100
  injury_risk ENUM('low', 'medium', 'high'),
  equipment_needed JSON           -- ["barra", "anillas"]
)

-- Progreso de usuario por skill
user_skill_progression (
  id INT PRIMARY KEY,
  user_id INT,
  skill_id INT,
  
  -- Estado actual
  current_step INT DEFAULT 0,     -- En qué paso de progressions va (0=locked)
  status ENUM('locked', 'unlocked', 'training', 'completed', 'mastered'),
  
  -- Métricas personales
  best_attempt_date TIMESTAMP,
  best_reps INT,
  best_hold_seconds INT,
  best_weight DECIMAL(5,2),
  
  -- Historial
  total_practice_sessions INT DEFAULT 0,
  total_practice_minutes INT DEFAULT 0,
  failed_attempts INT DEFAULT 0,
  
  -- Desbloqueos
  unlocked_at TIMESTAMP,
  completed_at TIMESTAMP,
  mastered_at TIMESTAMP,          -- Cuando domina el skill (ej: 10 reps seguidas)
  
  -- Notas
  personal_notes TEXT,
  last_feedback TEXT              -- Feedback del último intento grabado
)

-- Evaluaciones/exámenes
calisthenia_assessments (
  id INT PRIMARY KEY,
  user_id INT,
  assessment_type ENUM('level_test', 'skill_test', 'benchmark'),
  
  -- Info general
  scheduled_date DATE,
  completed_date TIMESTAMP,
  status ENUM('pending', 'completed', 'cancelled'),
  
  -- Resultados
  overall_score INT,              -- 0-100
  passed BOOLEAN,
  
  -- Detalle por ejercicio (JSON)
  results JSON,                   -- [
                                  --   {"exercise": "Pull-ups", "result": 12, "required": 10, "passed": true},
                                  --   {"exercise": "Muscle-up", "result": 0, "required": 1, "passed": false}
                                  -- ]
  
  -- Decisión
  recommended_action ENUM('level_up', 'continue', 'regress', 'rest'),
  notes TEXT,
  
  -- Próximos pasos
  next_assessment_date DATE
)
```
