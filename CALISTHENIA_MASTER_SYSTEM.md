# SISTEMA MASTER DE CALISTENIA
## Arquitectura de Progresión a Largo Plazo (2-5 años)

---

## 1. ESTRUCTURA DE NIVELES (12 Niveles)

No solo 3 niveles, sino **12 rangos progresivos** con evaluaciones:

```
NIVEL 1:  Novato Absoluto      (0-3 meses)     - Base cero
NIVEL 2:  Principiante I       (3-6 meses)     - Dominio básico
NIVEL 3:  Principiante II      (6-9 meses)     - Volumen inicial  
NIVEL 4:  Principiante III     (9-12 meses)    - Preparación intermedio
NIVEL 5:  Intermedio I         (1-1.5 años)    - Skills básicos
NIVEL 6:  Intermedio II        (1.5-2 años)    - Fuerza unilateral
NIVEL 7:  Intermedio III       (2-2.5 años)    - Skills intermedios
NIVEL 8:  Avanzado I           (2.5-3 años)    - Especialización
NIVEL 9:  Avanzado II          (3-3.5 años)    - Skills avanzados
NIVEL 10: Avanzado III         (3.5-4 años)    - Maestría técnica
NIVEL 11: Experto              (4-5 años)      - Elite calistenia
NIVEL 12: Master               (5+ años)       - Innovador/Atleta
```

## 3. EVALUACIONES Y BENCHMARKS

### Evaluación de Subida de Nivel (cada 3 meses)
Tests estandarizados que deben superarse:

**NIVEL 1→2:**
- 10 push-ups perfectos
- 30s plank
- 20 air squats
- 5 australian pull-ups

**NIVEL 4→5 (Principiante→Intermedio):**
- 5 pull-ups estrictos
- 15 push-ups
- 1 pistol squat asistido
- 20s L-sit

**NIVEL 7→8 (Intermedio→Avanzado):**
- 10 pull-ups
- Muscle-up
- Pistol squat completo
- 10s tuck front lever
- 3 HSPU en pared

**NIVEL 11→12 (Experto→Master):**
- One-arm pull-up
- Planche 5s
- Freestanding HSPU
- Front lever 10s
- Muscle-up anillas +10kg

## 4. PERIODIZACIÓN (Macrociclos)

### ESTRUCTURA ANUAL (52 semanas)

**FASE 1: BASE (Semanas 1-12)** - Hipertrófia/Acumulación
- 3-4 sets x 10-15 reps
- Frecuencia: 3x semana
- Enfoque: Volumen, técnica, conexión mente-músculo
- Skills: Fundamentos, progresiones iniciales

**FASE 2: FUERZA (Semanas 13-24)** - Intensidad
- 4-5 sets x 5-8 reps
- Frecuencia: 4x semana (Upper/Lower)
- Enfoque: Fuerza máxima, skills intermedios
- Skills: Negatives, holds intermedios

**FASE 3: SKILLS (Semanas 25-36)** - Especificidad
- 5-6 sets x 3-6 reps
- Frecuencia: 4-5x semana (Push/Pull/Legs)
- Enfoque: Skills, técnica avanzada
- Skills: Muscle-ups, levers, handstands

**FASE 4: PEAK/Deload (Semanas 37-44)** - Competición/Test
- Intensidad variable, testeos
- Frecuencia: 3-4x semana
- Enfoque: Mantenimiento, evaluación

**FASE 5: RECUPERACIÓN (Semanas 45-52)** - GPP/Movilidad
- Volumen bajo, movilidad, prehab
- Frecuencia: 2-3x semana
- Preparación siguiente macrociclo

## 5. SISTEMA DE PUNTOS/EXPERIENCIA

Cada entrenamiento otorga XP según:
- Volumen total (sets x reps)
- Nivel de dificultad
- Nuevos PRs (personal records)
- Skills desbloqueados
- Consistencia (streaks)

**Niveles desbloquean:**
- Nuevos ejercicios
- Nuevas rutinas
- Variaciones avanzadas
- Combinaciones complejas

## 7. FLUJO DE USUARIO

### Onboarding:
1. Evaluación inicial (test de 15 min)
2. Asignación nivel inicial (1-4)
3. Selección objetivos (fuerza/skills/peso)
4. Generación Plan Personalizado (12 semanas)

### Uso Diario:
1. Dashboard muestra nivel actual y XP
2. Sesión asignada según fase del ciclo
3. Registro de reps/sets (o completar skill)
4. Subida automática de nivel si pasa benchmark
5. Desbloqueo de nuevos ejercicios

### Evaluaciones:
1. Cada 4 semanas: Mini-test
2. Cada 12 semanas: Evaluación oficial
3. Si pasa → Subida nivel + Nuevos desafíos
4. Si no → Ajuste progresión + Continúa fase

## 8. TIPOS DE RUTINAS POR FASE

### FASE BASE (Volúmen)
- Full body 3x semana
- 4 ejercicios x 3 sets x 12-15 reps
- 60-90s descanso
- Enfoque: Forma perfecta, rango completo

### FASE FUERZA
- Upper/Lower 4x semana
- 5 ejercicios x 4-5 sets x 5-8 reps
- 2-3 min descanso
- Enfoque: Intensidad, progresión carga

### FASE SKILLS
- Push/Pull/Legs 4-5x semana
- Skills 20min + Strength 40min
- 4-5 ejercicios x 5 sets x 3-6 reps
- Enfoque: Técnica compleja, frecuencia

### FASE PEAK
- Split específico según objetivo
- Intensidad máxima, volumen moderado
- Testeos semanales
